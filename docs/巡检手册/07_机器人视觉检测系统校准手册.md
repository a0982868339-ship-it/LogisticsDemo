## 1. 校准目的

视觉检测系统的精度直接决定货物识别准确率和安全告警可靠性。本手册规定三类机器人视觉传感器（RGB相机、热成像仪、激光雷达）的定期校准方法、校准标准及判定准则，确保传感器性能始终满足EMIS系统的精度要求。

**校准触发条件：**
- 定期校准（按维护周期）
- 设备更换后校准
- 碰撞/跌落事故后校准
- 算法工程师判断精度下降后校准
- 系统年度大检时全量校准

## 2. RGB相机校准

### 2.1 校准设备要求

| 设备 | 规格 | 备注 |
|---|---|---|
| 标准棋盘格标定板 | 9×6格 / 格间距30mm / 精度±0.05mm | A2尺寸，配套防反光涂层 |
| 色卡（ColorChecker）| X-Rite 24色块标准色卡 | 用于白平衡和色彩一致性校准 |
| 标准照明箱 | 均匀光照强度800~1000 Lux / 色温5000K / 显色指数≥95 | D50标准光源 |
| 校准软件 | EMIS-CalibTool v3.x（内网地址：`calib.syzbq.internal`）| 需登录认证 |
| 对比度测试板 | ISO 12233分辨率测试板 | 用于MTF分辨率检测 |

### 2.2 内参标定流程

适用于：每季度例行校准、相机更换后。

**操作步骤：**

1. 将机器人停放于标准校准台（充电桩旁边的校准区，地面有红框标识，编号：CALIB-ZONE-XX）
2. 开机进入**校准模式**（EMIS-CalibTool > 相机校准 > 内参标定）
3. 按软件提示，将棋盘格标定板在相机前以**16种不同角度/距离**摆放（软件实时指引，绿色方框提示剩余姿态）：
   - 距离范围：0.5m ~ 3.0m
   - 水平偏转：-45° ~ +45°（步长15°）
   - 垂直偏转：-30° ~ +30°（步长15°）
4. 每个姿态按 F 键采集一帧，采集完16帧后点击【开始标定】
5. 系统自动计算焦距（fx, fy）、主点（cx, cy）、畸变系数（k1, k2, p1, p2, k3）
6. 校准结果判定：

| 参数 | 合格标准 | 不合格处置 |
|---|---|---|
| 重投影误差（RMSE）| ≤ 0.5 像素 | 重新采集，若仍超标则更换相机 |
| 焦距变化量（相比出厂值）| < ±3% | 记录，若>5%则告警并联系制造商 |
| 径向畸变k1 | \|k1\| < 0.15 | 严重偏离则检查镜头是否损伤 |

7. 校准通过后，新参数自动加载并写入机器人配置数据库（带时间戳，版本化管理）
8. 在EMIS-CalibTool中生成**校准报告**（PDF格式），上传至文档管理系统

### 2.3 外参标定（多相机联合）

**适用：** 相机更换或位置发生物理变化后。

外参标定确定各相机之间、相机与激光雷达之间的相对位姿关系（旋转矩阵R、平移向量T）。

```
操作步骤：
1. 在校准区域放置3个以上AprilTag标记板（Tag25h9族，边长20cm）
2. 在EMIS-CalibTool中选择【外参标定】模式
3. 启动多相机同步采集（硬件帧同步，触发精度±1ms）
4. 采集12个不同位置的标记板数据
5. 系统自动求解TF变换树：
   - camera_front → base_link
   - camera_rear → base_link
   - lidar_main → base_link
   - camera_thermal → base_link
6. 验证方式：在RViz中叠加显示各传感器数据，目测对齐精度
7. 验证通过（RGB与点云对齐误差≤2cm）后，写入外参配置文件
```

## 3. 热成像仪校准

### 3.1 非均匀性校正（NUC）

**触发时机：** 开机后自动执行（约90秒），环境温度变化>5°C时自动重触发。

| 步骤 | 操作 | 参数 |
|---|---|---|
| NUC-01 | 关闭镜头盖（机器人内置闸门自动关闭）| 时长：5秒 |
| NUC-02 | 采集均匀暗场参考帧 | 帧数：16帧均值 |
| NUC-03 | 打开镜头盖 | — |
| NUC-04 | 系统计算每个像素的偏置和增益校正系数 | 精度要求：噪声等效温差NETD ≤ 50mK |

### 3.2 温度绝对精度校准

**频率：** 每半年一次（需返回维护间使用黑体炉）

| 工具 | 规格 |
|---|---|
| 高精度黑体炉 | SR800N型 / 辐射面积150×150mm / 温度精度±0.03°C / 范围-20°C~+150°C |
| 参考温度计 | PT1000铂电阻，4线制，精度±0.05°C |
| 校准距离 | 1.0m（固定，使用光学定位台架）|

**校准点及合格标准：**

| 黑体设定温度 | 热成像读数合格范围 | 不合格阈值 |
|---|---|---|
| -10°C | -10°C ± 1.5°C | 偏差>2°C |
| 0°C | 0°C ± 1.0°C | 偏差>1.5°C |
| 25°C（室温）| 25°C ± 0.5°C | 偏差>1.0°C |
| 50°C | 50°C ± 1.0°C | 偏差>1.5°C |
| 100°C | 100°C ± 1.5°C | 偏差>2.0°C |

若任一点超出不合格阈值，须更换热成像仪或联系FLIR授权维修。

## 4. 激光雷达校准

### 4.1 外部目标点精度校准

**工具：**
- 高精度角反射器（Corner Reflector）/ 已知尺寸的标准靶板（1m×1m平面，精度±1mm）
- 全站仪（徕卡 TS16，角度精度0.5″，测距精度0.6mm+1ppm）

**步骤：**

1. 在校准区域布设**5个**基准角反射器，由全站仪精确测量其三维坐标（作为Ground Truth）
2. 启动激光雷达，采集60秒点云数据（60帧取均值）
3. 在EMIS-CalibTool中提取角反射器点云质心
4. 计算激光雷达坐标与全站仪坐标的**RMS误差**

| 距离 | 合格标准（RMS）| 超标处置 |
|---|---|---|
| 0~10m | ≤ 3cm | 重新校准；若持续超标，联系速腾聚创 |
| 10~30m | ≤ 5cm | 同上 |
| 30~100m | ≤ 10cm | 同上 |

### 4.2 时间同步精度检测

激光雷达点云数据须与IMU、相机帧时间戳精确同步（PPS秒脉冲 + GPS授时）：

| 参数 | 要求值 |
|---|---|
| 激光雷达时间戳绝对精度 | ±1ms（相对GPS时间）|
| 各传感器时间戳最大偏差 | ≤ 5ms（相互之间）|
| PPS信号误差 | ≤ 100ns |

## 5. 校准数据管理

| 要素 | 规定 |
|---|---|
| 校准记录格式 | JSON + PDF校准报告 |
| 记录保存位置 | EMIS文档管理系统 > 维护记录 > 校准报告 |
| 记录保存期限 | 设备服役期内 + 3年 |
| 校准证书有效期 | 依各传感器校准周期（见附表）|
| 校准人员资质要求 | 须持有园区颁发的《EMIS传感器校准授权证书》|
| 校准结果发布 | 校准通过后，系统自动发布新参数，通知AI算法组评估模型影响 |

## 7. 校准相关错误码

| 错误码 | 描述 | 处置措施 |
|---|---|---|
| CAL-E001 | 内参标定重投影误差超标（>0.5px）| 重新采集16姿态；若仍超标联系制造商 |
| CAL-E002 | 热成像NUC失败 | 手动触发NUC；失败则派单热成像组 |
| CAL-E003 | 激光雷达精度超标 | 检查设备是否受到碰撞，申请外部标定机构复验 |
| CAL-E004 | 校准软件连接机器人失败 | 检查网络连通性，确认机器人IP白名单 |
| CAL-E005 | 校准证书过期 | 立即安排重新校准，期间降低该传感器数据权重 |
| CAL-E006 | 外参标定RGB与点云对齐误差>2cm | 重新采集，检查标记板是否移动 |
