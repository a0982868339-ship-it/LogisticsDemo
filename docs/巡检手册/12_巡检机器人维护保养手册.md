## 4. 维护操作规程

### 4.1 开始维护前准备（通用）

**每次维护前必须执行：**

1. **确认锁定**：登录EMIS调度系统，将该机器人状态设置为"维护锁定"（MAINTENANCE LOCK），防止系统派发新任务
2. **物理断电确认**：按下机器人左侧紧急停止按钮（红色），确认机器人完全静止后方可进行任何接触
3. **危化品区额外要求**：佩戴适当防护装备后，由安全员陪同进场
4. **记录维护前状态**：在维修单系统（MAINT-SYSTEM）中开单，记录开始时间、维护类型、负责人
5. **工具清洁**：确认工具无油脂污染（避免污染光学传感器）

### 4.2 镜头清洁操作规程（SOP-MAINT-101）

```
操作步骤：
1. 将机器人置于维护锁定状态
2. 用压缩空气罐吹除镜面大颗粒灰尘（角度：30°斜吹，距离5cm）
   ⚠️ 禁止直吹镜面中心，防止划伤
3. 用一滴相机镜头清洁液滴在镜头纸上（非滴在镜头上）
4. 轻轻以圆周方向擦拭镜面，从外到内螺旋状
   ⚠️ 切勿用力按压，切勿来回擦拭
5. 用干净镜头纸干擦一次，去除残留清洁液
6. 拍摄测试图像（在EMIS-CalibTool中【镜头清洁验证】模式），
   检查图像中心区域无油渍、划痕
7. 如有残留污渍，重复3-6步骤（最多3次）
8. 填写维修单，标注"镜头清洁"，关闭工单
```

### 4.3 驱动轮更换操作规程（SOP-MAINT-201）

所需工具：
- 轮毂拆卸专用扳手（P/N: TOOL-X7-001）
- 扭矩扳手（量程：0~100N·m）
- 备用驱动轮（P/N: HY-X7-TW-001）

| 步骤 | 操作 | 要点 |
|---|---|---|
| 01 | 维护锁定，机器人抬升（液压举升台，举升高度15cm）| 确认举升台额定载重≥250kg |
| 02 | 拆下轮毂盖（4颗M8螺钉，扭矩25N·m）| 螺钉保存，避免丢失 |
| 03 | 拆下固定螺母（M20，扭矩120N·m，左驱反牙）| 注意左右驱动方向 |
| 04 | 安装新驱动轮，对准定位孔（3个）| 确认卡口完全咬合 |
| 05 | 按标准扭矩（120N·m）紧固固定螺母 | 扭矩扳手核验 |
| 06 | 安装轮毂盖，紧固螺钉（25N·m）| — |
| 07 | 降低液压台，机器人落地 | — |
| 08 | 在维护场区（无人）进行前后直行5m测试 | 观察无跑偏 |
| 09 | 填写维修单，标注新轮胎批次号 | — |

### 4.4 电池更换操作规程（SOP-MAINT-301）

> ⚠️ **磷酸铁锂电池存在挤压/穿刺起火风险，必须由EMIS-MaintCert-L2持证人员操作！**

电池规格核对（更换前确认）：

| 参数 | 要求值 |
|---|---|
| 电池型号 | HY-BAT-X7-200AH |
| 标称电压 | 48V |
| 容量 | 200Ah（9,600Wh）|
| 重量 | 约82kg |
| 认证标志 | UN38.3 / CE / CCC |
| 制造日期 | 距当前日期≤12个月 |

操作要点：
- 佩戴绝缘手套（1000V级），使用绝缘工具

## 5. 故障诊断指南

### 5.1 常见机械故障

| 故障现象 | 可能原因 | 诊断方法 | 处置措施 |
|---|---|---|---|
| 直行偏移（单侧）| 左右轮磨损不均 / 驱动电机功率不均 | 比较左右轮磨损量；检查电机电流 | 更换磨损轮；调整电机参数 |
| 异常噪音（驱动轮）| 轮毂螺母松动 / 轴承损坏 | 手动检查螺母扭矩；振动分析仪检测 | 重新紧固或更换轴承 |
| 爬坡力不足 | 电机功率下降 / 减速器油脂少 | 检查电机电流（额定值：15A/路）| 补充油脂或更换电机 |
| 充电对接失败 | 充电触点氧化 / 停靠偏差 | 清洁触点；校准停靠精度 | 触点清洁；调参 |
| 急停不灵敏 | 急停按钮气门弹簧老化 | 手动测试急停响应时间（应≤100ms）| 更换急停按钮组件 |

### 5.2 常见软件/系统故障

| 错误码 | 故障描述 | 诊断步骤 | 处置措施 |
|---|---|---|---|
| MAINT-E001 | 导航栈崩溃（ROS2进程退出）| 查看`/var/log/ros2/nav.log`崩溃堆栈 | 重启导航栈；若频繁则报告算法组 |
| MAINT-E002 | 定位漂移（EKF发散）| 检查IMU传感器数据频率是否正常（目标：100Hz）| 重置EKF；若持续则IMU校准 |
| MAINT-E003 | 摄像头掉线（USB/CSI断开）| 检查连接器松动；运行`ros2 topic hz /camera/image` | 重新插拔连接器；驱动重启 |
| MAINT-E004 | 本地存储写满（>95%）| `df -h`检查存储使用率 | 手动清理已上传的过期原始数据；扩容 |
| MAINT-E005 | 时间戳不同步（NTP偏差>50ms）| `chronyc tracking`查看NTP状态 | 强制同步：`chronyc makestep` |
| MAINT-E006 | 5G模块掉线 | 检查`/dev/ttyUSB0`设备存在性；`mmcli -m 0`状态 | 重启5G模块；若仍失效则更换模块 |
| MAINT-E007 | 温控风扇停转（整机过热）| 检查风扇供电和控制信号 | 手动关机散热；更换风扇 |
