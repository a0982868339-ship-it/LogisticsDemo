# 人员协同作业安全规范

**文件编号：** SYZBQ-SEC-016  
**版本号：** V2.0.0  
**发布日期：** 2025年04月01日  
**编制部门：** 安全消防大队 / 智能运营中心  
**审批人：** 副总经理（主管运营）徐国梁  
**密级：** 内部资料

---

## 1. 目的

三亚综合保税区同时存在大量人员（日均约6.2万人次在园）与372台具身智能巡检机器人的混合作业环境。本规范旨在明确人机协同作业的安全边界、行为准则、事故预防及处置要求，保障所有人员的人身安全，避免机器人设备损坏。

---

## 2. 人机安全边界规定

### 2.1 安全距离要求

| 场景 | 机器人类型 | 人员最小安全距离 | 超限处置 |
|---|---|---|---|
| 机器人正常行进中 | X7Pro | 0.8m | 机器人减速至0.3m/s |
| 机器人正常行进中 | X5 | 0.6m | 同上 |
| 机器人正常行进中 | M3（室外）| 1.5m | 机器人减速，启动语音警告 |
| 机器人作业（静止检查中）| 任何机型 | 1.0m（环形）| 语音提示：「作业中，请勿靠近」 |
| 机器人高速段（专属通道）| 任何机型 | 2.0m | 专属通道两侧设导引护栏 |
| 危化品区机器人 | X7Pro（Ex）| 2.0m（额外危险）| 该区域限制人员进入 |

### 2.2 行为准则（全体人员）

**禁止行为：**

1. **禁止在机器人行进路线前方突然横穿**（每年可能导致意外碰撞）
2. **禁止触碰或骑坐机器人**（违者处200元罚款，情节严重暂停入园资格）
3. **禁止遮挡机器人传感器**（遮挡激光雷达、相机等）
4. **禁止在机器人充电桩附近无关停留**（充电区地面标注黄色警示线）
5. **禁止使用强光手电直接照射机器人相机镜头**
6. **禁止逃避机器人采集**（故意躲避、戴面罩等行为，须配合调查）
7. **禁止强行打开机器人检修门或拆卸零件**（须立即报告安全部）

**鼓励行为：**
- 发现机器人异常时及时拨打内线 98-2100 报告
- 在机器人减速提示时，主动让出机器人行进路线
- 配合机器人进行人员管控区域确认（如刷工牌）

## 3. 人机冲突场景处置

### 3.1 机器人传感器识别到人员时的自动行为

| 距离 | 机器人响应 | 响应时间 |
|---|---|---|
| < 2.0m（正常作业区）| 降速至0.6m/s，调整局部路径 | 50ms内 |
| < 0.8m（X7Pro）| 降速至0.3m/s，请求人员让路（语音）| 50ms内 |
| < 0.5m（任何机型）| 安全制动停止（0.5m/s²减速度）| 50ms内 |
| < 0.3m（危险接近）| 紧急制动（2.0m/s²减速度）+ 报警 | 50ms内 |

### 3.2 机器人发生轻微碰撞时

**处置流程：**

1. 机器人自动停止，启动碰撞传感器日志记录
2. 当事人（如有受伤）立即拨打120（内线1120）
3. 安全员到场，保护事故现场，收集机器人视频片段（自动保留碰撞前后60秒视频）
4. 通知智能运营中心，禁止在事故鉴定完成前恢复该机器人作业
5. 填写《人机碰撞事故报告》（FORM-SAFE-001），24小时内提交安全主管
6. 事故原因分类：
   - **人员违规**（违反安全距离规定）→ 教育培训，视情节处理
   - **机器人传感器故障**（传感器未正确检测到人员）→ 技术故障处理，机器人停用排查
   - **环境因素**（光线不足、地面湿滑等）→ 改善作业环境

## 4. 特殊区域人员管理

### 4.1 危化品区（ZONE-E）

| 规定项目 | 要求 |
|---|---|
| 进入资质 | 须持危化品从业资格证（或陪同有资质人员进入）|
| 进入手续 | 登记姓名、工号、进入时间、预计停留时间（系统登记+纸质备份）|
| 防护要求 | 最低要求：安全鞋+安全帽+防化手套；HZ-A区须B级以上防化服 |
| 禁止携带物 | 非防爆手机（ZONE-E内须使用防爆手机）；火机、打火机 |
| 最大同时在场人员 | HZ-A区：≤6人；HZ-B区：≤12人；HZ-C区：≤20人 |
| 机器人与人员安全间距 | 2.0m（机器人主动避让）|
| 人员监测 | 每人佩戴气体检测个人报警仪（LEL报警阈值：10%）|

### 4.2 高价值货物专区（ZONE-D）

| 规定项目 | 要求 |
|---|---|
| 进入权限 | 须在系统中有ZONE-D访问授权（合规审计部审批）|
| 进入手续 | 双重身份验证（工牌刷卡+人脸识别）|
| 禁止携带物 | 大型箱包（统一存放门口储物柜）；私人手机（限使用园区提供无拍摄手机）|
| 机器人在场期间 | 鼓励人员配合机器人作业，1m内请不要遮挡机器人视线 |
| 人员行为记录 | 全时段多角度录像，保存180天 |

### 4.3 冷链区域（ZONE-C，低温环境）

| 安全要求 | 规定 |
|---|---|
| 进入冷冻区(-18°C)防护 | 须穿戴保暖工作服（园区统一配发，记录领用）|
| 单次低温区作业时间 | CK-A区(-25°C)：≤45分钟连续作业，休息至少30分钟后续次 |
| 低温区机器人电池充电 | 低温型机器人须在过渡区（-5°C区）充电，禁止在深冷区直接充电 |
| 低温区急救设备 | 每个冷库区门口备有应急复温毯和热饮（定期更换）|

## 5. 新员工人机安全培训要求

| 培训课程 | 内容 | 时长 | 频率 | 考核要求 |
|---|---|---|---|---|
| EMIS系统人机安全基础 | 机器人行为规则、安全距离、禁止行为 | 2小时 | 入职时必修 | 笔试≥80分通过 |
| 危化品区安全规程 | 进入资质、防护要求、应急处置 | 4小时 | 需进入ZONE-E人员必修 | 笔试+实操 |
| 冷链区安全操作 | 低温防护、紧急情况处置 | 2小时 | 需进入ZONE-C人员必修 | 笔试≥80分 |
| 高价值区规范 | 进入程序、行为规范 | 1小时 | 需进入ZONE-D人员必修 | 笔试 |
| 年度安全复训 | 全部内容复习，新增规定培训 | 2小时 | 每年1次（全员）| 签字确认 |
